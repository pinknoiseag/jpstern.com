# Hostpoint: absoluter Pfad zum Webroot (für AuthUserFile in .htaccess)
#
# Per SSH einloggen, in das Verzeichnis wechseln wo index.html liegt:
#   cd ~/www
#   pwd
#
# jpstern.com (bestätigt):
#   ~/www/jpstern.com
#   /home/hpsterm/www/jpstern.com
#
# In public/.htaccess ist AuthUserFile bereits gesetzt auf:
#   /home/hpsterm/www/jpstern.com/.htpasswd
#
# Falls pwd auf dem Server einen anderen Benutzernamen zeigt, nur diesen
# einen Pfad in .htaccess anpassen und neu builden.
